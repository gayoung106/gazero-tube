## Gazero_tube(유튜브 프로젝트)

## 설정

```
npx create-react-app ./
```

## 실행

```
npm start
```

## 라이브러리

```
npm i @emotion/react
npm i @emotion/styled
npm i @mui/icons-material
npm i @mui/material
npm i axios
npm i react-player
npm i react-router-dom
```

cf) 최신버전 업데이트

```
npm install --legacy-peer-deps
```
